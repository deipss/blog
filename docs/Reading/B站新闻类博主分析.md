# 风险的预警

其实在做全球的实讯方面的内容，信息方面的风险是很多的，包括；
- 1 内容版权方面的权益
- 2 信息合规方面的内容（政治敏感、内容暴力等）

除了最主要的两方面的内容风险以外，还有外部的同行竞争、恶意举报等等。

2025-6-30就被小红书下架了3个视频，原因是【信息并未公布】
